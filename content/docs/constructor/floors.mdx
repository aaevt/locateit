---
title: Этажи
description: Создание, удаление и переключение между этажами
---

Система этажей позволяет:

- Создавать неограниченное количество этажей
- Хранить каждый этаж в виде JSON
- Переключаться между этажами без потери данных

## API

Этажи управляются через `useFloorCanvasManager`.

## UI

Панель этажей позволяет переключаться, переименовывать и удалять этажи.
