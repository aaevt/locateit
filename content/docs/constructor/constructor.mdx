description: Инструмент для создания интерактивных карт помещений
---

Добро пожаловать в конструктор! Здесь вы можете создавать планы этажей, добавлять стены, двери, лестницы и многое другое.

## Возможности

- Множественные этажи с сохранением состояния
- Кастомизация объектов: комнаты, стены, двери
- Интуитивный UI и управление через канвас
- Поддержка экспорта и импорта JSON
